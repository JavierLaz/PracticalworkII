What is Machine Learning?

Machine learning is a subset of artificial intelligence (AI) that enables systems to learn from data and make predictions or decisions without explicit programming. It focuses on developing algorithms that identify patterns, make decisions, and improve over time.

Unlike traditional programming, which uses data and fixed rules to produce answers, machine learning uses the seeked results and vasts amount of data to produce rules. These rules are then implemented on a model capable of giving the answers based on the input data.

Types of Machine Learning

  • Supervised Learning: Algorithms learn from labeled data to make predictions.
  • Unsupervised Learning: Algorithms identify patterns in unlabeled data.
  • Reinforcement Learning: Algorithms learn by interacting with an environment and receiving feedback.

Prerequisites

Before starting to develop machine learning models, it is crucial to have a solid foundation on some concepts of mathematics, data science and computer science.

Mathematics and Statistics:

Mathematics are the foundation of machine learning, the data structures and algorithms used in machine learning are all the result of algebra, calculus and statistics.

  • Linear Algebra

    Linear algebra is applied on every machine learning by the use of techniques like

    • Principal component analysis
    • Linear Regression
    • Suport vector machines
    • Neural Networks
    • Jacobian Matrix
  • Calculus

    • Calculus is the tool used in machine learning to minimize the error of a model. A crucial aspect that allows machine learning models to be functional. Some key calculus concepts that need to be studied are:

    • Multivariable optimization
    • Jacobian Matrix
    • Gradient Descent
    • Linear Regression
    • Logistic Regression
    • Approximation methods
  • Statistics

    Statistics refers to the science of collecting, organizing, analyzing, interpreting, and presenting data. Therefore statistics are

    • Predictive algorithms throgh inferential statistics
    • Data Analysis through descrptive statistics
    • Data visualization
    • Anomaly detection

Programming skills:

Programming skills are necessary to implement the theoretical mathematical concepts into a machine learning model.Therefore, being able to use tools like Python and R is mandatory.

  • Python

    Python is one of the most popular programming languages in machine learning due to it's high performance libraries for data management like:

    • Numpy
    • Pandas
    • Tensorflow
  • R

    R is a powerful tool used in the data analysis and data visualization field it is accessible due to it's comprehensive data functions and because R is an open source language.

    • ggplot2
    • data.table
